简易和声编写工具 HARMOLOID 1.0
作者 科林/sdercolin

HARMOLOID是一款免费的制作简单和声的工具，主要面向VOCALOID、UTAU等音声合成软件的用户。

※使用方法
	请参阅"用户手册.html"。

※运行环境
若程序无法运行，请下载安装Microsoft .NET Framework 2.0或以上版本：http://www.microsoft.com/zh-cn/download/details.aspx?id=1639
